
# Sudoku Solver Project

## Project Overview
This project is a Sudoku Solver built using HTML, CSS, and JavaScript.

It uses the Backtracking Algorithm to solve Sudoku puzzles automatically.

---

## Features
- Beautiful modern UI
- Responsive design
- Fast Sudoku solving
- Input validation
- Sample puzzle loader

---

## Files Included
- `index.html` → Main webpage
- `style.css` → UI styling
- `script.js` → Sudoku solving logic
- `README.md` → Documentation

---

## How to Run

### Method 1 (Easy)
1. Extract the ZIP file.
2. Open the folder.
3. Double-click `index.html`.

The project will open in your browser.

### Method 2 (VS Code)
1. Open the folder in VS Code.
2. Install the "Live Server" extension.
3. Right-click `index.html`.
4. Click "Open with Live Server".

---

## Algorithm Used
The project uses the Backtracking Algorithm:

1. Find an empty cell.
2. Try numbers 1 to 9.
3. Check if valid.
4. Continue recursively.
5. Backtrack if invalid.

Time Complexity: Approximately O(9^(n*n))
